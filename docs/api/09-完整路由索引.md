# 09. 完整路由索引

> 说明：`Auth` 列含义
> - `Public`：无需登录
> - `JWT`：需要 `Authorization: Bearer ...`
> - `JWT+Approved`：需要登录且账号审核通过
> - `Admin`：管理员
> - `Admin/Owner`：管理员或群主
